Thesis

Akamai is the legacy king of internet content delivery — 4,400 edge locations across 700+ cities globally — being repriced as the infrastructure layer for the agentic AI internet. The declining CDN business is being eclipsed by two high-growth segments: cybersecurity (API security >$100M run rate, growing >100% YoY) and cloud infrastructure services (CIS at ~$400M run rate, guided 45–50% growth in 2026).

The catalyst: Akamai Inference Cloud — the first global-scale implementation of NVIDIA's AI Grid reference design, deploying Blackwell GPUs across the existing edge network. Jensen Huang named Akamai alongside AWS at GTC 2026 as among the first to operationalize AI Grid. At 3.8x P/S vs. Cloudflare at 33x, the market is still pricing AKAM as a declining CDN company. If CIS inflects, the re-rate toward 6–8x P/S implies $175–230+.

4P Framework

People
Co-founder Tom Leighton (MIT applied math professor) has led AKAM since 2013. Credible technologist, but execution on the cloud pivot has been slow. CFO Ed McGowan has been transparent about margin pressure during the transition. Insider selling is not elevated — a positive signal during a heavy investment year.
Product
Three-pillar business: (1) Security — API security, WAF, bot management, DDoS, Zero Trust; adds $200–250M/yr in new revenue with renewal rates in the 90s%. (2) Cloud Infrastructure Services — compute, storage, Kubernetes, GPU inference via Linode acquisition + NVIDIA partnership; ~$400M run rate. (3) Legacy CDN — declining but stabilizing with fewer competitors post-Edgio collapse. The 4,400-location physical edge network is a genuine moat that hyperscalers cannot replicate overnight.
Potential
CIS guided 45–50% growth in 2026. Edge AI inference TAM expected to grow from $54B (2024) to $157B (2030). Akamai's network uniquely positions it for latency-sensitive inference workloads — gaming, fintech, media, robotics, autonomous driving. The $200M four-year enterprise contract validates real demand. If pipeline converts, CIS alone could be $600M+ run rate by year-end 2026.
Predictability
Mixed. Security is highly predictable (90%+ renewal rates, recurring). CIS is accelerating but early-stage with revenue recognition lagging contract signings. Legacy CDN provides cash flow but is in secular decline. The "investment year" narrative means margins compress before they expand. Stock dropped 9% after Feb 2026 earnings on conservative guidance despite strong segment growth.

Key Catalyst

GTC 2026 — NVIDIA Endorsement

Akamai Inference Cloud was highlighted in Jensen Huang's GTC 2026 keynote as the first global-scale implementation of NVIDIA AI Grid. Akamai and AWS were named as the first cloud service providers to offer RTX PRO 4500 Blackwell Server Edition instances. Being named alongside the world's largest cloud provider for an NVIDIA reference architecture is a significant credibility upgrade for a company the market still prices as a legacy CDN player. The Inference Cloud deploys across 4,400 edge locations with semantic caching, intelligent workload routing, and NVIDIA Dynamo/NeMo/NIMs integration.

Valuation Context

Cloudflare (NET) trades at 33x P/S with negative GAAP earnings. DigitalOcean (DOCN) at 8.4x after a 59% YTD run. Fastly (FSLY) at 6x with insiders dumping into a 145% squeeze. AKAM at 3.8x with $400M cloud run rate growing 45–50%, $200M+ security additions annually, and an NVIDIA co-signed edge inference platform. The valuation gap reflects legacy CDN overhang — but the business mix is shifting faster than the multiple.
